To run the project:

1. Go to "Builds/Windows/build/bin/x64/Master"
2. Run "MAT500Project3" application.

To see code:

1. In root folder, open MAT500Project3.sln
2. To import into Unity, open new Unity project and import unitypackage called "MAT500Project3", located in root folder.

Note:

In C#, Vector3 uses floats by default, which I could not change to doubles.
Due to this, at a high number of points like 20, the curve does not exactly overlap with the last point.
